About BinBeacon

Website Introduction

BinBeacon is a map-based application designed to help users find and locate trash bins in area around Neili (currently). Whether you're at XingRen park, YuanZe university, or dormitory, BinBeacon helps you find the nearest trash can in a matter of seconds. The application also provides additional features like customizing the map style, filtering trash bins by type, and adding new bins to the map, you can even check and update the status of each bin!

Key Features

  • Interactive Map: View a map with markers showing the locations of nearby trash bins. Try clicking on the markers!
  • Live Location Marker: See your current location on the map in real-time, helping you find the nearest trash can with ease.
  • Filter Options: Customize the map view by filtering trash bins based on categories like recyclable, non-recyclable, or both.
  • Add New Bins: Users can add new trash bins to the map, which will be stored in our backend database and displayed for others to see.
  • Map Reset: If you scroll away from your location, you can easily reset the map view to center on your current location.

Technologies Used

The website was built using the following technologies:

  • Frontend: HTML, CSS, JavaScript, Leaflet.js (for interactive maps)
  • Backend: Node.js, Render
  • Database: Supabase
